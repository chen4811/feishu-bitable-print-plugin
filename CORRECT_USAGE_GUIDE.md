# 🚨 飞书插件使用错误及正确方法

## ❌ 你遇到的问题

```
该BlockTypeID对应的小组件不支持当前方式开发。
如需调试其他小组件，请前往"项目详情"修改BlockTypeID

BlockTypeID: blk_698d829dea010cbd4f294012
CLI ID: cli_a9073e804b39dcb2
```

### 问题原因

你在飞书开发者工具中尝试使用 **BlockTypeID** 开发方式来加载插件，但**飞书多维表格边栏插件不支持这种方式**！

## ✅ 正确的使用方法

### 🔥 方法 1：通过飞书开放平台（推荐）

**这是唯一正确的方法！**

#### 步骤 1：访问飞书开放平台
```
https://open.feishu.cn
```

#### 步骤 2：创建应用
1. 登录你的飞书账号
2. 点击「应用管理」
3. 点击「创建应用」
4. 选择「自建应用」
5. 填写应用信息并创建

#### 步骤 3：创建插件
1. 进入应用详情页面
2. 在左侧菜单找到「插件管理」
3. 点击「创建插件」
4. **重要**：选择插件类型 → **多维表格 - 边栏插件**
5. 填写插件信息：
   - 插件名称：飞书多维表格排版打印插件
   - 插件描述：支持多维度排版、批量打印、二维码/条形码生成、PDF导出等
6. 点击「创建」

#### 步骤 4：上传插件包
1. 在插件详情页面
2. 点击「上传插件」
3. 选择文件：`feishu-bitable-plugin-v2.0.0.zip`
4. 等待上传完成
5. 点击「保存」

#### 步骤 5：配置权限
1. 进入「权限管理」
2. 申请权限：
   - `bitable:app:read` - 读取表格数据
   - `bitable:app:write` - 写入表格数据
3. 点击「申请」

#### 步骤 6：提交审核
1. 返回「插件管理」
2. 确认配置无误
3. 点击「提交审核」
4. 等待审核通过（1-3 个工作日）

#### 步骤 7：在飞书表格中使用
1. 审核通过后，打开任意飞书多维表格
2. 点击右上角「...」菜单
3. 选择「插件」
4. 找到你的插件并点击
5. 开始使用！

## ❓ 常见问题

### Q: 为什么不能用飞书开发者工具？

A: 飞书开发者工具的 BlockTypeID 开发方式是为小程序和小组件设计的，不是为多维表格边栏插件设计的。边栏插件必须通过飞书开放平台的插件管理功能开发。

### Q: 如何测试插件？

A: 插件上传并通过审核后，在飞书多维表格中打开插件进行测试。

### Q: 审核需要多长时间？

A: 通常 1-3 个工作日。

### Q: 如何加快审核？

A: 确保插件包完整（包含 icon.png），配置正确，描述清晰。
