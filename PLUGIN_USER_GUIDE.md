# 飞书多维表格排版打印插件 - 使用指南

## 🎯 插件功能

### 核心功能

#### 1. 📊 数据展示
- **多视图支持**：表格、看板、卡片、时间轴四种视图
- **实时搜索**：支持按任务名称和描述搜索
- **数据统计**：显示记录数量、最后更新时间等信息

#### 2. 🖨️ 排版打印
- **记录选择**：单选、多选、全选要打印的记录
- **字段映射**：自定义要打印的字段及显示名称
- **打印模板**：4种预设模板（标准卡片、紧凑卡片、简洁列表、标签模式）
- **布局配置**：自定义列数、纸张大小、页面方向、字体大小等
- **打印预览**：实时预览打印效果
- **PDF 导出**：一键导出为 PDF 文件

#### 3. 🔲 条码生成
- **二维码**：单个/批量生成二维码
- **条形码**：支持 CODE128、CODE39、EAN13、UPC 等格式
- **自定义内容**：可选择字段作为条码内容

## 🚀 快速开始

### 方式一：本地开发测试

```bash
# 1. 启动开发服务器
pnpm dev

# 服务器将在 http://localhost:5000 运行
```

### 方式二：在飞书中使用

1. **打开**任意飞书多维表格
2. **点击**右上角的「插件」按钮
3. **选择**「自定义插件」
4. **点击**「+新增插件」
5. **输入**运行地址：
   - 本地测试：`http://localhost:5000`
   - 生产环境：`https://your-plugin.vercel.app`（需先部署）
6. **点击**「确定」

## 📱 使用说明

### 步骤 1：查看数据

在主页面，你可以：
- 切换不同视图（表格、看板、卡片、时间轴）
- 搜索特定记录
- 查看统计信息

### 步骤 2：进入排版打印

点击右上角的「排版打印」按钮，进入打印配置页面。

### 步骤 3：选择记录

在左侧面板中：
- **全选**：勾选"全选"复选框
- **单选**：点击单个记录卡片
- **多选**：点击多个记录卡片

### 步骤 4：配置打印

在右侧配置面板中：

#### 4.1 选择模板
- 标准卡片：适合打印任务卡片
- 紧凑卡片：适合批量打印
- 简洁列表：表格形式展示
- 标签模式：适合打印标签、贴纸

#### 4.2 配置字段
- 勾选要打印的字段
- 自定义字段显示名称
- 拖拽调整字段顺序（即将推出）

#### 4.3 设置布局
- **每行卡片数**：1-4 列
- **纸张大小**：A4、A5、Letter
- **页面方向**：纵向、横向
- **字体大小**：9pt-14pt
- **显示边框**：是/否
- **显示页眉**：自定义页眉文本
- **显示页脚**：自定义页脚文本

#### 4.4 高级选项
- **二维码**：为每条记录生成二维码
- **条形码**：为每条记录生成条形码
- **自动序号**：为打印内容添加序号

### 步骤 5：预览与打印

- **预览**：点击「预览」按钮查看打印效果
- **打印**：点击「打印」按钮调用浏览器打印
- **导出 PDF**：点击「导出PDF」按钮生成 PDF 文件

## 🎨 功能详解

### 打印模板

#### 1. 标准卡片
- **适用场景**：任务卡片、工作证、信息卡
- **特点**：
  - 2 列布局
  - 显示边框
  - 包含页眉和页脚
  - 字体大小：12pt

#### 2. 紧凑卡片
- **适用场景**：批量打印、清单列表
- **特点**：
  - 3 列布局
  - 显示边框
  - 无页眉页脚
  - 字体大小：10pt

#### 3. 简洁列表
- **适用场景**：任务清单、汇总表
- **特点**：
  - 单列布局
  - 表格形式
  - 包含页眉和页脚
  - 字体大小：11pt

#### 4. 标签模式
- **适用场景**：标签、贴纸、座位牌
- **特点**：
  - 4 列布局
  - 显示边框
  - 无页眉页脚
  - 字体大小：9pt

### 字段映射

支持的字段类型：
- ✅ 文本字段
- ✅ 数字字段
- ✅ 单选字段
- ✅ 多选字段
- ✅ 日期字段
- ✅ 人员字段
- ✅ 复选框字段
- ✅ URL 字段
- ✅ 邮箱字段

### 条码生成

#### 二维码
- **内容来源**：可选择任意字段作为二维码内容
- **自动生成**：为每条记录自动生成唯一二维码
- **下载支持**：支持单独下载每个二维码

#### 条形码
- **支持格式**：
  - CODE128：通用格式，支持数字和字母
  - CODE39：工业标准格式
  - EAN13：商品条码（需要 13 位数字）
  - UPC：美国商品条码

## 🔧 高级功能

### 自定义打印样式

在打印预览模式下，可以使用浏览器打印功能进一步自定义：
1. 点击「打印」按钮
2. 在打印对话框中：
   - 选择纸张大小
   - 调整边距
   - 选择是否打印背景图形
   - 设置页眉页脚

### 批量导出 PDF

1. 选择要导出的记录
2. 配置打印模板
3. 点击「导出PDF」
4. 等待生成完成
5. 自动下载 PDF 文件

### 使用二维码/条形码

#### 在打印中包含二维码
1. 进入「选项」标签
2. 勾选「为每条记录生成二维码」
3. 选择二维码内容字段
4. 预览并打印

#### 在打印中包含条形码
1. 进入「选项」标签
2. 勾选「为每条记录生成条形码」
3. 选择条形码内容字段
4. 选择条形码格式
5. 预览并打印

## 📝 常见问题

### Q1: 为什么显示模拟数据？

**A**: 插件检测到不在飞书环境中，自动使用模拟数据。在飞书多维表格中添加插件后，将显示真实的表格数据。

### Q2: 如何获取真实的多维表格数据？

**A**: 
1. 将插件部署到 HTTPS 服务器（推荐 Vercel）
2. 在飞书多维表格中添加插件地址
3. 插件会自动获取当前表格的数据

### Q3: 打印时显示不完整怎么办？

**A**: 
- 尝试调整每行卡片数
- 减小字体大小
- 切换到横向打印
- 使用"简洁列表"模板

### Q4: PDF 导出失败怎么办？

**A**:
- 检查浏览器控制台错误信息
- 尝试减少记录数量
- 检查是否有特殊字符
- 刷新页面后重试

### Q5: 可以自定义打印模板吗？

**A**: 当前版本提供 4 种预设模板。自定义模板功能正在开发中，敬请期待！

## 🔐 隐私与安全

### 数据权限

- **前端插件**：权限与当前登录用户一致
- **数据存储**：不存储任何飞书数据
- **数据传输**：仅在浏览器中处理，不发送到外部服务器

### 授权码说明

- 前端自定义插件**不需要**授权码
- 授权码（PersonalBaseToken）仅用于服务端插件
- 当前插件为前端插件，无需授权码
