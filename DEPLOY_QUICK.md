# 🚀 一键部署到 Vercel

## 📋 前提条件

- GitHub 账号（用于 Vercel 登录）
- 5 分钟时间

---

## 🎯 快速部署（3 步）

### 步骤 1：执行部署命令

```bash
pnpm deploy:vercel
```

### 步骤 2：登录 Vercel（首次需要）

选择 **GitHub** 登录，按提示操作。

### 步骤 3：获取部署地址

部署成功后，你会看到：

```
✔ Production: https://your-plugin.vercel.app
```

这就是你的插件地址！

---

## 📱 在飞书中使用

1. 打开飞书多维表格
2. 点击「插件」→「自定义插件」
3. 点击「+新增插件」
4. 输入部署地址（如 `https://your-plugin.vercel.app`）
5. 点击「确定」

开始使用！
